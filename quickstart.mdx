Proyecto Colegio requires a PostgreSQL database and Node.js 18 or later. The steps below walk you through cloning the repository, wiring up the database, and launching the development server so you can explore the application locally.

<Note>
The `build` script in `package.json` runs `prisma generate && next build`. The `postinstall` hook also runs `prisma generate`, so the Prisma client is always regenerated after `npm install`.
</Note>

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js 18+** — [nodejs.org](https://nodejs.org)
- **PostgreSQL** — a local instance or a hosted database (Supabase, Neon, Railway, etc.)
- **Git** — [git-scm.com](https://git-scm.com)

## Run the app locally

<Steps>
  <Step title="Clone the repository">
    Clone the Proyecto Colegio repository to your local machine.

    ```bash
    git clone https://github.com/BapJ40/PROYECTO-COLEGIO.git
    cd PROYECTO-COLEGIO
    ```
  </Step>

  <Step title="Install dependencies">
    Install all project dependencies. The `postinstall` hook automatically runs `prisma generate` after installation.

    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Set up your environment variable">
    Create a `.env` file in the project root and add your PostgreSQL connection string.

    ```bash .env
    DATABASE_URL="postgresql://USER:PASSWORD@HOST:5432/DATABASE_NAME"
    ```

    Replace `USER`, `PASSWORD`, `HOST`, and `DATABASE_NAME` with your actual database credentials.
  </Step>

  <Step title="Push the schema to the database">
    Because there is no migrations folder, use `db push` to sync the Prisma schema directly to your database.

    ```bash
    npx prisma generate && npx prisma db push
    ```

    This creates all tables defined in `prisma/schema.prisma` — students, teachers, sections, subjects, attendance records, and study plans.
  </Step>

  <Step title="Start the development server">
    Run the Next.js development server.

    <CodeGroup>

    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    ```bash pnpm
    pnpm dev
    ```

    </CodeGroup>
  </Step>

  <Step title="Open the application">
    Open your browser and navigate to [http://localhost:3000](http://localhost:3000). The Proyecto Colegio dashboard loads once the database connection is established.
  </Step>
</Steps>
